# 📅 Planeador Semanal

Uma aplicação web progressiva (PWA) para organizar a tua vida por áreas e alcançar o equilíbrio.

## 🚀 Como Instalar

### Opção 1: GitHub Pages

1. **Fork este repositório** ou cria um novo repositório no GitHub
2. **Faz upload dos ficheiros:**
   - `index.html`
   - `manifest.json`
   - `icon-192.png`
   - `icon-512.png`

3. **Ativa o GitHub Pages:**
   - Vai a Settings → Pages
   - Em "Source", seleciona "main" branch
   - Clica em "Save"

4. **Acede ao teu planeador:**
   - Após 1-2 minutos, estará disponível em:
   - `https://teu-username.github.io/nome-do-repositorio`

### Opção 2: Adicionar ao Ecrã Principal

#### 📱 Android (Chrome)
1. Abre o link do teu planeador no Chrome
2. Toca nos 3 pontos (⋮)
3. Seleciona "Adicionar ao ecrã principal" ou "Instalar aplicação"
4. Confirma e pronto! ✅

#### 📱 iPhone (Safari)
1. Abre o link do teu planeador no Safari
2. Toca no botão Partilhar (seta para cima)
3. Seleciona "Adicionar ao Ecrã Principal"
4. Confirma e pronto! ✅

## ✨ Funcionalidades

- 📋 **3 Vistas diferentes:** Por Categoria, Agenda do Dia, Vista Geral
- 💾 **Guardado automático** de todos os dados
- 📱 **Funciona offline** depois de instalado
- 🎨 **7 Áreas personalizáveis:** Atividade Física, Formação, Profissional, Pessoal, M's, Casa, Outros
- ⏰ **17 slots horários** (06:00 - 22:00)
- 🔄 **Categorias arrastáveis** para reordenar
- 📝 **Notas por categoria**
- 📊 **Exportação para Word** (em desenvolvimento)

## 🛠️ Tecnologias

- React 18
- HTML5 + CSS3
- LocalStorage para persistência
- PWA (Progressive Web App)
